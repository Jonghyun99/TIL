@ 인간의 본성(인성론)

​

1. 성선설 : 맹자, 루소,주자, 양명, 다산

* 맹자 : 불인지심(차마 어쩌지 못하는 마음), 

*맹자의 사단과 사덕

측은지심/인 (불쌍하고 측은)

수오지심/의 (부끄러워하고 미워)

사양지심/예 (양보 사양)

시비지심/지 (옮고그름 판단 지혜)

- 왕도정치주장(=덕치사상)

- '인의'중시

- 인간의본성은 태어날때부터 선함

​

2. 성악설 : 순자, 홉스

* 순자 : 화성기위 : 인간의 본성은 선하게 교화(변화)가능(의지강조)

- 혼락을 극복하기 위해 '예' 중시

- 인간은 태어날때부터 악하나, 인위적 규제를 통한 교화가능(인간의 악한본성 규제 -> 질서유지)

​

3. 성무성악설 : 고자, 로크(백지설),에라스무스, 칸트, 듀이

- 인간의 본성은 정해져있지 않고 개인의 의지나 환경의 영향으로 변화

에라스무스 : 완성되지 않은 밀랍

로크 : 백지설

칸트: 칸트는 무조건적인 선이 무엇인지를 물음으로써 보편타당한 도덕 판단의 기준을 확보하고자 했다. 

칸트는 무조건성을 갖춘 유일한 선은 ( 선의지 ) 밖에 없다고 했다.

,듀이

* 정약용 성 기호설 : 인간은 태어날때 선악을 가지고태어나지않음, 선을 좋아하는 욕구(기호=경양성)을 가지고 태어남

5. 규범윤리학과 메타윤리학

​

ㅁ 규범윤리학(=이론윤리학)

- 보편적 윤리 규칙생성

* 의무론 : 인간이 마땅히 지켜야할 도덕법칙

  자연법 (이퀴니스 - 스토아)

  칸트의 사회계약설(행위중시)

* 공리주의 (=행위주의) : 결과중시

★ 양적공리 (벤담) : 쾌락은 한 종류, 외적제제:제제가능, 최대다수 최대행복을 도덕입법원리로 함

★ 질적공리 (밀) : 쾌락의 질적차이도 존재, 배고픈인간-소량의양질쾌락(정신적) > 배부른돼지-다량의저질쾌락

* 덕윤리 (아리스토텔레스) : 도덕법칙과 원리보다 행위자의 내면과 도덕성 중시

ㅁ 응용윤리학(실천윤리학) : 현실문제의 대안제시

​

ㅁ 메타윤리학(윤리학적 회의론 - 규범윤리비판)

- 윤리적 진술들을 분석하는 일만 하는 비철학적 철학(분석적 탐구)

- 분석윤리학

- 논리적이론 : 논리의 타당성 추론 - 논리가능성검토

- 의미론적 이론

​

※ 아리스토텔레스의 덕윤리

# 실존주의

- 자본주의의 발전에 따른 인간의 물신화 현상

- 전쟁의다발성, 과학기술의 발전이 원인이 됨

- 특징 : 인간소외현상 - 인간의 주체성강조

- 유신론적 실존주의 (키에르케고르) : 현실적 나의 삶을 자각하는데 신이 필요함

- 무신론적 (하이데거, 사르트스) : 인간의 자유와 주체성강조

​

1. 키에르케고르 : 나는나를아는가

- 주체성, 주관성 강조

- 현실속 존재하는 나의문제 탐구(실존)

- 유신론적 자유주의

​

2. 사르트스

- 무신론적 자유주의 (자유,책임)

- 피투성 (내던저짐)

- 자유의 부조리

- 현 존재 ( 현 존재 -> 거기있음)

- 인간은 죽음에 이르는 존재 (죽음을 직시할 경우 본래의 실존 탐구가능)

@ 기타윤리사상

​

1. 정의윤리 : 도덕성 -> 도덕적 추론, 판단능력

- 타인을 부당하게 대우하지 말라

- 추상적 보편적 도덕원칙의 공정한 적용 주장

- 개인의 존엄성과 권리보장

​

2. 평화윤리

- 소극적평화 : 비폭력상태 (물리적 폭력의 제거)

- 적극적평화 : 물리적 + 문화적 + 구조적 폭력의 소멸

​

3. 배려윤리 :  배려와 보살핌을 지향하는 여성의 본질적성향에 주목

- 타인의 고통에 대한 관심, 보살핌 중시

ㅁ 길리건 : 여성적입장에서 기존윤리비판

ㅁ 나딩스 : 길리건의 배려윤리 명료화

                 배려범위의 확대 -> 동식물, 지구환경

​

3. 책임윤리 : 현대사회에서 책읨의 주체가 불문명(익명성문제) -> 개인의 행위에 대한 책임감 강조

ㅁ 배버의 책임윤리

- 심정윤리 : 행위의 결과보다는 '선'한의도 중시

- 책임윤리 : 예견가능한 행위의 결과에 따른 엄격한책임

ㅁ 요나스의 책임윤리

- 과학문명이 초래한 위기극복방안

- 과학기술의 발달로 인한 기존윤리와의 공백 -> 윤리적공백

- 인간만이 책임을 질수있는 유일한 존재 (의무강조)

- 생태계로 책임범위 확장

@ 통일을 위한 평화비용

2. 분단비용 : 분단으로 인해 소요되는 무, 유형의 지출성비용 (소모적 비용)

- 무형적(간접적 비용) : 경제적 사회적 문화적 손실,  전쟁가능성에 대한 공포, 이산가족의 고통, 이념과 갈등의 대립

3. 통일비용 : 두 체제가 통합한 후 부담해야할 비용 (투자적 비용)

- ex. 제도통합비용, 위기관리비용, 경제적관리비
