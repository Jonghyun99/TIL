## 웹 시스템(web system)

웹 시스템 처리는 클라이언트 사이드와 서버 사이드로 구분된다.

## 클라이언트 사이드와 서버 사이드

웹 시스템에서의 처리는 클라이언트와 서버 중 어느 쪽에서 수행되는 작업인가에 따라 다음과 같이 구분할 수 있다.

1. 클라이언트 사이드(client-side)
2. 서버 사이드(server-side)

브라우저는 서버에서 수신한 HTML 문서를 해석하여 화면을 표시한다.

여기서 자주 사용하는 언어는 자바스크립트가 있다.

서버 사이드란 서버 측에서 수행하는 처리를 의미한다.

브라우저에서 요청한 HTML 문서에 서버 사이드 스크립트 언어가 포함되어 있으면, 서버에서 처리 한 후 브라우저에 보내게 된다.

여기에 들어가는 언어가 PHP이다.

이런 서버 사이드에서 가공되어 생성하는 페이지를 동적 웹 페이지 (dynamic web page)라고 한다.

## PHP(PHP: Hypertext Preprocessor)

PHP는 C언어를 기반으로 만들어진 서버 측에서 실행되는 서버 사이드 스크립트 언어이다.

PHP로 작성된 코드를 HTML 안에 추가하면, 웹 서버는 코드를 해석하여 문서를 실행한다.
