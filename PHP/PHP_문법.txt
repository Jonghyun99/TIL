## PHP 코드 영역

PHP 코드는 웹 페이지 어느 부분에나 위치할 수 있다.

따라서 PHP 파서가 PHP 코드를 인식하기 위해서는 그 시작과 끝을 알려줄 필요가 있다.

문법

```php
1. PHP 권장 스타일: <?php ... ?>
2. HTML 스크립트 스타일: <script language = "php"> ... </script>
3. SGML 스타일 : <? ... ?>
4. ASP 스타일: <% ... %>
```

서버 별 이식성 문제를 위해 PHP 스타일을 사용하는 것이 권장된다.

## PHP 문법

C 언어처럼 PHP 명령문도 마지막은 세미콜론으로 끝난다.

```php
<?php
	echo "Welcome to PHP Class";
?>
```

PHP 코드가 종료되면 자동으로 세미콜론이 적용되어 세미코론을 빼먹어도 정상적으로 동작한다.

```php
<?php
	echo "Welcome to PHP Class"
?>
```

아래와 같이 종료태그를 생략할 수 있다.

```php
<?php
	echo "Welcome to PHP Class";
```

그렇지만 항상 세미콜론과 종료태그를 사용해주자.

## PHP 주석

```php
1. 한 줄 C 언어 스타일: //주석문
2. 여러 줄 C 언어 스타일: /*주석문*/
3. 한 줄 쉘 스타일: #주석문
```

아래와 같은 상황에서는 정상적인 주석처리가 되지 않아 주의하여야 한다.

```php
/* 여러 줄

/* 또 다른 여러 줄 주석입니다. */

주석입니다. */
```

## echo() 함수

echo()는 C언어의 printf() 같이 HTML 스트림에 문자열을 출력해준다.

echo()는 실제 함수가 아니므로, 인수가 전달될 때 괄호를 사용해도 되고, 생략할 수 있다.

또한, 가변 길이 인수와 같이 함수 문맥으로 사용할 수 없다.

하지만 두개 이상의 인수를 전달할 때에는 괄호를 사용하면 안된다.

PHP는 키워드, 클래스, 함수의 대소문자를 구분하지 않아 Echo, EcHo 모두 같은 동작을 한다.

```php
<?php
    echo "문자열을 출력합니다.<br>";
    echo ("인수를 전달할 때 괄호를 사용해도 괜찮습니다!<br>");
    ECHO "echo() 함수의 키워드는 대소문자를 구분하지 않습니다!<br>";
    echo "첫 번째 인수, ", "두 번째 인수";
    //echo("첫 번째 인수, ", "두 번째 인수"); // 오류 발생
?>
```
