## 웹 프로그래밍 언어

웹 시스템 중 서버 측에서 실행되는 서버 사이드 언어를 웹 프로그래밍 언어라고도 부른다.

이러한 웹 프로그래밍 언어 중 많이 사용되는 언어는 다음과 같다.

1. PHP
2. JSP
3. ASP.NET

예전에는 많은 상업용 사이트가 PHP를 사용했지만, 보안 취약성으로 인해 JSP나 ASP.ET으로 대체되고 있다.

## PHP 동작 원리

PHP는 웹 시스템인 클라이언트 / 서버 환경에서 동작하는 서버 사이드 스크립트 언어이다.

브라우저 → 웹서버 → PHP 파서 → 데이터베이스 → PHP 파서 → 웹 서버  → 브라우저

1. 웹페이지 요청
2. 스크립트 실행 요청
3. 데이터베이스 작업
4. 작업 처리 결과
5. 웹페이지 응답
